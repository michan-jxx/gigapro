# GIGA PRO
PHP自作
## 概要
教育センター職員や学校に勤務されている職員をターゲットとしたWEBアプリを作成しました。<br>
学校で生徒に配布しているタブレットを管理できるものになっております。<br>
教育センター職員と各学校の先生方それぞれログイン画面を分け、それぞれでログインできるようにしています。
## 使い方
#### 教育センター職員
